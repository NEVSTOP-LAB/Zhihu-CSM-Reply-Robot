# AI-009 实施记录：主流程 run_bot.py — MVP 版

## 任务目标
串联所有模块的主入口，支持 pending/ 模式。

## 实施内容

### 1. 模块实现 (`scripts/run_bot.py`)
- **load_config()**: 加载 settings.yaml + articles.yaml
- **load_seen_ids() / save_seen_ids()**: 增量处理状态持久化
- **write_pending_reply()**: 生成 YAML front-matter + Markdown 的 pending 文件
- **process_article()**: 完整处理流程
  - 拉取评论 → 检测真人回复 → 前置过滤 → 超长截断 → RAG检索 → LLM生成 → pending/
  - 真人回复自动索引到 reply_index（高权重）
  - 更新 seen_ids 增量标记
- **main()**: 主入口
  - 初始化所有模块
  - 遍历文章列表处理
  - ZhihuAuthError / BudgetExceededError → 告警
  - 连续失败 → 告警
  - 退出前保存状态

### 2. 遇到的问题
- **dedup 过滤误触发**: 同一作者的多条评论在测试中被 dedup 过滤
  - 解决方案: 测试中使用不同作者 + reset_dedup_cache()
