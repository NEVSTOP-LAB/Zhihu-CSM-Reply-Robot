# AI-002 实施记录：项目初始化与目录脚手架

## 任务目标
建立项目骨架，使后续模块可独立开发。

## 实施内容

### 1. 目录结构创建
- `scripts/` — 所有 Python 模块（含 `__init__.py`）
- `config/` — 配置文件
- `data/` — 运行时数据（seen_ids.json, wiki_hash.json）
- `tests/` — 测试目录（含 `conftest.py`）
- `archive/articles/` — 归档目录
- `pending/` — 人工审核目录
- `csm-wiki/` — CSM Wiki 文档目录
- `docs/实施记录/` — 实施记录目录

### 2. 配置文件
- `config/articles.yaml` — 监控文章列表（2 个示例条目）
- `config/settings.yaml` — 全局配置（bot/llm/rag/vector_store/review/filter/alerting 全部配置段）

### 3. 模块骨架
以下模块均创建了含 docstring 和类型注解的骨架文件：
- `scripts/run_bot.py` — 主入口
- `scripts/zhihu_client.py` — 知乎 API 封装
- `scripts/rag_retriever.py` — RAG 检索
- `scripts/llm_client.py` — LLM 调用
- `scripts/thread_manager.py` — 对话线程管理
- `scripts/comment_filter.py` — 评论过滤
- `scripts/archiver.py` — 归档写入
- `scripts/wiki_sync.py` — Wiki 同步
- `scripts/alerting.py` — 告警模块
- `scripts/cost_tracker.py` — 费用追踪

### 4. 其他文件
- `requirements.txt` — 依赖列表
- `.gitignore` — 忽略规则（含 vector_store, reply_index, .env）
- `data/seen_ids.json` — 初始状态文件
- `data/wiki_hash.json` — 初始哈希文件
